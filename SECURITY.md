# Security Policy

## Supported Versions

The following versions of `zvec-rs` are currently supported with security updates:

| Version | Supported          |
|---------|-------------------|
| latest  | ✅ Yes            |
| < latest| ❌ No             |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Please do not open a public issue for security vulnerabilities.**

Instead, report it privately via GitHub Security Advisories:

1. Go to the repository's **Security** tab.
2. Click **Report a vulnerability** to open a private security advisory.

We will acknowledge receipt within 48 hours and aim to provide an initial assessment within 5 business days. We will work with you to understand the issue, develop a fix, and coordinate disclosure.

## Disclosure Policy

We follow a coordinated disclosure approach:

1. The vulnerability is reported privately.
2. We investigate and develop a fix.
3. A patch release is prepared.
4. The fix is released, and the advisory is published.

We ask that you give us a reasonable amount of time to address the issue before disclosing it publicly.
